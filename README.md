Athletik_marion
===============

Cette application web utilise Symfony.

Il vous faut installer un environnement de développement pour Symfony soit
- installer composer
- installer apache
- installer php 7
- installer (ou wamp)
- afin d'avoir un dossier vendor, il faudra installer un nouveau projet symfony et de l'écraser en clonant le projet dedans
- modifier le parameters.yml afin d'y mettre les informations de votre base de données
- démarrer le serveur par les lignes dans votre projet $php bin/console server:run

N'hesitez pas à vous réferer à la documentation officielle de l'installation de Symfony

Enjoy!
